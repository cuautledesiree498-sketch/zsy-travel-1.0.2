## 当前替换规则

### Hero 首屏
优先级：
1. Sanity 后台「首页头图区 > 背景图」
2. Sanity 后台「网站全局设置 > 默认首页背景图」
3. 本地默认图 `public/media/hero/hero-mountains-city.jpg`

### Hero 视频
优先级：
1. Sanity 后台「首页头图区 > 背景视频链接」
2. 若无视频，则显示图片

### 目的地卡片图
优先级：
1. Sanity 后台对应卡片里的「背景图」
2. 本地目的地默认图

## 后续你怎么换

### 换首页首屏图
Sanity 后台：
- 首页搭积木配置
- 打开「首页头图区」
- 上传「背景图」
- Publish

### 换首页首屏视频
Sanity 后台：
- 首页搭积木配置
- 打开「首页头图区」
- 修改「背景视频链接」
- Publish

### 换目的地卡片图
Sanity 后台：
- 首页搭积木配置
- 打开「目的地卡片区」
- 点开某一张卡片
- 上传「背景图」
- Publish

## 备注

当前素材仅用于第一阶段快速上线与视觉占位。
